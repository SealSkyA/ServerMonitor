# 设置页滚动与配置备份需求

## 简介

设置页支持滚动，并提供服务器配置和主题设置的导出与导入功能。

## 术语

- **配置备份**：包含备份版本、导出时间、服务器配置和主题模式的 JSON 文件。
- **导入恢复**：使用备份中的服务器配置替换当前服务器列表，并应用备份主题模式。

## 需求

### 需求 1：设置页滚动

**用户故事：** 作为应用用户，我希望设置页可以垂直滚动，以便访问全部设置项。

#### 验收标准

1. 当设置内容高度超过可视区域时，设置页应支持垂直滚动。
2. 设置页应保留底部导航栏所需的内容安全间距。

### 需求 2：导出配置

**用户故事：** 作为服务器管理员，我希望导出配置备份，以便在设备更换或恢复时使用。

#### 验收标准

1. 当 Android 用户选择导出配置时，系统应打开系统文件管理器的保存位置选择界面。
2. 配置备份应包含备份版本、导出时间、服务器配置和主题模式。
3. 系统应生成精确到毫秒的默认备份文件名。
4. 当用户取消保存位置选择时，系统应保留当前配置且不显示成功提示。

### 需求 3：导入配置

**用户故事：** 作为服务器管理员，我希望导入配置备份，以便恢复服务器列表和主题设置。

#### 验收标准

1. 当用户选择有效配置备份时，系统应展示导入数量确认信息。
2. 当用户确认导入时，系统应替换当前服务器列表并应用备份主题模式。
3. 当用户选择格式无效的文件时，系统应展示导入失败提示。
