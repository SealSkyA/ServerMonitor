# 文件面板工作流需求

## 需求

### 目录排序

当用户进入任意目录时，系统应先显示文件夹，再按名称显示普通文件和压缩包。

### 创建入口

当用户选择创建文件或文件夹时，系统应显示带有类型标签和明确边框的名称输入框。

### 上传

当用户选择文件或文件夹上传时，系统应显示上传进度弹窗。用户可将上传任务转入后台，系统应在应用内状态栏显示实时进度并继续传输。

### 全屏文件面板

当用户点击文件面板全屏按钮时，系统应在终端页面内显示全屏文件管理界面，并提供返回终端的入口。

### 编辑器返回

当编辑器处于打开状态且用户触发 Android 返回键时，系统应关闭编辑器并显示终端界面。

### 删除确认

当用户选择删除文件或文件夹时，系统应显示居中确认弹窗，展示目标名称与类型，并在用户确认后执行删除操作。
