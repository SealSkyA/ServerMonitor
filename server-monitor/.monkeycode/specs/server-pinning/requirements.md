# 服务器置顶需求

## 需求

### 服务器置顶

当用户选择置顶服务器时，系统应将服务器保存为置顶状态，并在服务器列表顶部显示该服务器。

### 置顶折叠

当服务器列表存在置顶服务器时，系统应显示置顶分组及数量。用户可折叠或展开置顶分组。

当置顶服务器处于展开状态时，系统应在置顶服务器与普通服务器之间显示浅色分隔控制条。分隔控制条应提供折叠置顶服务器操作。

当置顶服务器处于折叠状态时，系统应在普通服务器列表前显示浅色分隔控制条。分隔控制条应提供展开置顶服务器操作。

分隔控制条应使用与服务器卡片一致的圆角、半透明背景、细边框和间距层级。

### 连接反馈

当用户发起服务器连接时，系统应在连接入口显示加载动画并禁用重复连接操作，直到连接请求结束。
