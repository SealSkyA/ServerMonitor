# 服务器置顶设计

Feature Name: server-pinning
Updated: 2026-07-17

## 设计

服务器数据增加可选 `pinned` 字段并沿用现有持久化存储。列表按置顶状态分组，置顶组提供本地折叠状态。展开的置顶组在最后一个置顶服务器后渲染分隔控制条；折叠的置顶组在普通服务器前渲染同一控制条。控制条使用与服务器卡片一致的圆角、半透明背景、细边框、阴影和横向内边距，包含左侧置顶图标、中间操作文案、数量和右侧方向箭头。连接请求使用每台服务器的连接中状态，列表按钮与操作菜单读取同一状态并显示旋转加载图标。

## 验证

- 置顶状态在应用重启后保留。
- 展开时置顶服务器显示于普通服务器前方。
- 折叠时显示置顶分组标题与数量。
- 连接过程中对应入口显示加载状态。
