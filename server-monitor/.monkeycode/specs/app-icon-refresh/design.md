# 应用图标更新设计

Feature Name: app-icon-refresh
Updated: 2026-07-17

## 描述

Android 启动图标采用重新设计的服务器监控图形：深色底板承载紧凑服务器机架与观察镜头组合。自适应图标前景增加 14dp 内边距，保证桌面启动器的裁切区域完整展示图形。

## 组件与接口

- `mipmap-*`：提供标准、圆形与前景启动图标位图。
- `mipmap-anydpi-v26`：引用启动图标前景资源的自适应图标定义。

## 正确性属性

- 所有图标变体使用相同参考图形。
- Android 资源名称保持现有清单引用不变。

## 测试策略

- 执行 Android 调试构建。
- 在 Android 启动器中确认图标视觉。
