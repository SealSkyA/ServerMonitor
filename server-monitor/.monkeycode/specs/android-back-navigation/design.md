# Android 返回导航设计

Feature Name: android-back-navigation
Updated: 2026-07-17

## 设计

应用路由层注册唯一的 Capacitor `backButton` 监听器。监听器先派发可取消的浏览器事件，终端编辑器可通过取消事件关闭文件编辑器或退出全屏。未被拦截的事件按页面路径导航至父页面；主页面依据 Capacitor 的 `canGoBack` 信息返回浏览器历史或调用 `App.exitApp`。

## 验证

- 终端编辑器打开时，物理返回键关闭编辑器。
- 服务管理页的物理返回键进入对应服务器详情页。
- 服务器详情页的物理返回键进入服务器列表页。
- 主页面的物理返回键退出应用或返回浏览器历史。
