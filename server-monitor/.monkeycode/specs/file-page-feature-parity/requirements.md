# 文件页面功能同步需求

## 简介

独立文件页面保留现有界面样式，并具备终端文件面板提供的文件创建、文件夹上传、传输进度和文件编辑能力。

## 术语

- **独立文件页面**：应用底部导航中的“文件”页面。
- **终端文件面板**：终端页面中显示的文件浏览区域。
- **大文件分页**：每次读取固定大小的连续文件区块。

## 需求

### 需求 1：文件操作能力

**用户故事：** 作为服务器管理员，我希望在独立文件页面创建文件和文件夹，以便直接管理远程目录。

#### 验收标准

1. 当用户选择新建文件时，系统应在当前远程目录创建指定名称的文件。
2. 当用户选择新建文件夹时，系统应在当前远程目录创建指定名称的文件夹。
3. 当创建操作完成时，系统应刷新当前目录列表。

### 需求 2：文件夹上传与状态

**用户故事：** 作为服务器管理员，我希望从独立文件页面上传本地文件夹并查看进度，以便管理目录树。

#### 验收标准

1. 当用户选择上传文件夹时，系统应将文件夹层级上传至当前远程目录。
2. 当文件夹上传进行时，系统应展示文件数量和百分比进度。
3. 当文件夹上传完成时，系统应刷新当前目录列表。

### 需求 3：文件编辑器

**用户故事：** 作为服务器管理员，我希望在独立文件页面编辑远程文本文件，以便维护服务器配置。

#### 验收标准

1. 当用户选择编辑文件时，系统应打开包含文件内容的编辑器。
2. 当用户保存常规文件时，系统应将编辑内容写回原始远程路径。
3. 当文件超过单次读取区块大小时，系统应以分页只读预览展示连续区块。
4. 当用户在编辑器中搜索文本时，系统应显示匹配数量并支持上一个和下一个匹配项。

### 需求 4：文件操作菜单

**用户故事：** 作为服务器管理员，我希望长按文件或文件夹后在居中菜单中选择操作，以便在双面板间管理远程文件。

#### 验收标准

1. 当用户长按文件或文件夹时，系统应显示居中的两列操作菜单。
2. 当用户选择复制或移动时，系统应将另一个文件面板作为目标并显示目标面板方向。
3. 当复制、移动、压缩、解压、保存或修改属性执行时，系统应显示居中的进度弹窗直至操作完成。

### 需求 5：压缩与属性

**用户故事：** 作为服务器管理员，我希望配置压缩文件并查看文件属性，以便维护远程文件。

#### 验收标准

1. 当用户选择压缩时，系统应显示文件名、格式、压缩级别与密码表单。
2. 当用户选择属性时，系统应显示名称、目录、类型、大小、修改时间、权限、所有者和用户组。
3. 当用户在属性中选择更多时，系统应提供文件权限和修改时间操作。

### 需求 6：顶栏工具与书签抽屉

**用户故事：** 作为服务器管理员，我希望通过顶栏工具管理当前目录，并从文件页底栏上滑打开书签抽屉，以便快速返回常用远程路径。

#### 验收标准

1. 当用户打开顶栏三点菜单时，系统应将菜单锚定在右上角并覆盖三点按钮。
2. 当用户选择全选当前目录时，系统应仅选中活动面板当前目录中的全部文件和文件夹。
3. 当用户在文件页底栏向上滑动时，系统应打开书签抽屉并显示已收藏路径。
4. 当用户在书签抽屉中选择书签时，系统应在活动面板打开书签对应的远程路径。
5. 当用户在书签抽屉菜单中添加分组时，系统应保存分组名称并展示分组。
6. 当用户在书签抽屉菜单中调整高度时，系统应在半屏和近全屏高度之间切换。
7. 当用户新增分组时，系统应在“书签”标签后追加分组标签并将分组顺序保存。
8. 当用户长按分组标签时，系统应展示左移、右移、重命名和删除操作。
9. 当用户进入远程目录时，系统应保存最近访问记录并在“最近”标签中展示记录。
10. 当用户打开书签抽屉时，系统应播放自底部进入的展开动画。
