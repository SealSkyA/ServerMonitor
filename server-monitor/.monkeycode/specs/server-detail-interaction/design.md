# 服务器详情交互设计

Feature Name: server-detail-interaction
Updated: 2026-07-17

## 设计

详情页使用全高独立纵向滚动容器，滚动位置变化超过 8px 时更新底部导航的可见状态。应用路由层接收自定义事件，并为底部导航应用位移动画，同时在隐藏状态移除主内容的底部导航预留。操作栏使用横向滚动、触摸横向手势、指针拖动、过度滚动隔离和滚动吸附。文件与服务页面同样使用独立纵向滚动容器。系统信息使用单列信息行。

## 验证

- 详情页内容可上下滚动。
- 上滑隐藏底部导航，下滑显示底部导航。
- 操作栏可通过触摸水平滚动。
- 运行时长和温度各占一行。
